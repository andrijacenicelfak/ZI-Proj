# ZI-Proj
Projekat iz predmeta Zastita Informacija
Implementacija algoritama za kriptovanje podataka.
Algoritmi : 
  - Bifid
  - Knapsack
  - RC6
  - RC6 - CRT
  - Tiger hash
 
Implementacija web servisa koji koristi ove algoritme za komunikaciju sa klijentima.
Tehnologije : .NET 7
